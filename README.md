# MotleyTech Website

Marketing homepage for MotleyTech LLC, built with Next.js 15, React 19, and Tailwind CSS 4 for deployment on Vercel.

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm run start
```

## Deployment

Import this repository into Vercel and use the default Next.js settings. Vercel will install dependencies from `package.json` and run `npm run build`.
